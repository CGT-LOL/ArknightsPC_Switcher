明日方舟PC版 换服工具 (ArknightsPC Switcher)
通过文件覆盖的方式，实现一个客户端，双服启动。你不再需要下载两个游戏客户端以及另外一个服务器的游戏文件，只需几秒钟即可完成服务器切换，便利B服玩家和双修玩家
---
原理
对Windows 文件资源管理器的 复制/粘贴 操作自动化
不涉及任何内存注入或数据包修改
理论上封号风险极低（等同于你手动替换文件），但请自行承担风险
